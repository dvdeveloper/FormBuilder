FormBuilder
===========

Crea formulario dinámicamente con sus propiedades y obtén un debug completo de cada componente en formato JSON.
Componentes permitidos:

1)	Input text
2)	Textarea
3)	Select (dropdown)
4)	Checkbox
5)	Radio Button


Plugins
--------------------------------------
1. Jquery
2. bootstrap
